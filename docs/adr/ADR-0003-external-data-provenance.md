# ADR-0003: Proveniência e retenção de dados externos

- Status: Aceito
- Data: 2026-07-15

## Decisão

Todo dado externo registra provedor, captura, atualização, validade e confiança. A política do provedor define retenção, cache, expiração, atualização e atribuição.

Dados criados pelo usuário, como CRM, favoritos, observações, histórico, tarefas e configurações, são permanentes até ação explícita do usuário.
