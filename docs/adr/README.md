# Architecture Decision Records

Este diretório é o registro oficial das decisões arquiteturais do ProspectAI.

Uma decisão aceita não deve ser alterada silenciosamente. Mudanças futuras devem criar um novo ADR que substitua explicitamente o anterior, preservando o histórico.

## Registros aprovados

- ADR-0001 - Gateway seguro para chaves sensíveis.
- ADR-0002 - Deduplicação por níveis de confiança.
- ADR-0003 - Proveniência e retenção de dados externos.
- ADR-0004 - Separação entre pontuação determinística e IA.
- ADR-0005 - ADRs como documentação evolutiva obrigatória.
- ADR-0006 - Aplicativo Android local-first e banco como fonte de verdade.
- ADR-0007 - Favorito como atributo independente do status do CRM.
- ADR-0008 - Atualizações por APK assinado e migrações preservando dados.
- ADR-0009 - Injeção por construtor e composition root explícito.
- ADR-0010 - Retenção externa aplicada por manutenção periódica.
- ADR-0011 - Integration Manager com adaptadores, cache e observabilidade.
- ADR-0012 - Identificadores oficiais externos preservados por provedor.
- ADR-0013 - Catálogo de empresas paginado, filtrável e ordenável no banco.
- ADR-0014 - Ativação de provedor condicionada à política de armazenamento.
- ADR-0015 - Retenção aplicada na inicialização, no backup e no Worker diário.
